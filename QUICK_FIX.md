# Быстрое исправление: Загрузка аватара и редактирование профиля

## Проблема
Не работает загрузка аватара и редактирование email в профиле.

## Решение за 30 секунд

### Шаг 1: Остановите backend сервер
Нажмите `Ctrl+C` в терминале где запущен backend

### Шаг 2: Запустите миграцию
```bash
cd backend
python migrate_db.py
```

### Шаг 3: Запустите backend снова
```bash
python app.py
```

### Шаг 4: Обновите страницу в браузере
Нажмите `F5` или `Ctrl+R` (Cmd+R на Mac)

## Готово! ✅

Теперь вы можете:
- ✅ Загружать аватары (JPG, PNG, GIF до 5 МБ)
- ✅ Редактировать имя, местоположение и информацию о себе
- ⚠️ Email остается неизменяемым (это сделано для безопасности)

## Что было исправлено?

Добавлены недостающие колонки в базу данных:
- `avatar` - для хранения пути к аватару
- `bio` - для информации о себе
- `location` - для местоположения

## Альтернативный способ (автоматический)

Просто перезапустите backend - миграция применится автоматически:
```bash
cd backend
python app.py
```

При запуске вы увидите:
```
✓ Добавлена колонка 'avatar' в таблицу users
✓ Добавлена колонка 'bio' в таблицу users
✓ Добавлена колонка 'location' в таблицу users
✓ База данных успешно инициализирована
```

---

**Примечание:** Email нельзя изменить после регистрации. Это сделано для безопасности аккаунта. Если вам нужно изменить email, создайте новый аккаунт.
