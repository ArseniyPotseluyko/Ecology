# Темная тема

## Описание

Добавлена полноценная темная тема с переключателем день/ночь и автоматическим сохранением предпочтений пользователя.

## Возможности

### 1. Переключатель темы 🌓
- **Расположение**: В шапке сайта рядом с переключателем языка
- **Дизайн**: Анимированный переключатель с иконками ☀️ (день) и 🌙 (ночь)
- **Анимация**: Плавный переход между темами (0.3s)

### 2. Автоматическое сохранение ✅
- Предпочтения сохраняются в `localStorage`
- Тема восстанавливается при следующем посещении
- Ключ хранения: `ecology_theme`

### 3. Цветовая схема темной темы 🎨

**Основные цвета:**
- Фон первичный: `#0f172a` (темно-синий)
- Фон вторичный: `#1e293b` (серо-синий)
- Фон третичный: `#334155` (светло-серый)
- Текст первичный: `#f1f5f9` (почти белый)
- Текст вторичный: `#cbd5e1` (светло-серый)
- Зеленый акцент: `#22c55e` (яркий зеленый)

### 4. Охват компонентов 📦

Темная тема применяется ко всем компонентам:
- ✅ Header (шапка)
- ✅ Hero (главный баннер)
- ✅ About (о проекте)
- ✅ Services (направления деятельности)
- ✅ Reserves Map (карта заповедников)
- ✅ Contact (контакты)
- ✅ Footer (подвал)
- ✅ Reserve Detail (детали заповедника)
- ✅ Weather Widget (виджет погоды)
- ✅ Account (личный кабинет)
- ✅ Ecology (экология)
- ✅ Activity Detail (детали направлений)
- ✅ Booking (бронирование)

## Технические детали

### Архитектура

**ThemeContext** (`src/ThemeContext.js`):
- React Context для управления темой
- Хук `useTheme()` для доступа к теме
- Автоматическое сохранение в localStorage
- Применение класса `dark-theme` к body

**ThemeToggle** (`src/components/ThemeToggle.js`):
- Компонент переключателя
- Анимированный UI
- Доступность (aria-label)

**Стили** (`src/theme.css`):
- CSS переменные для цветов
- Глобальные стили для темной темы
- Плавные переходы между темами

### Использование в коде

```javascript
import { useTheme } from '../ThemeContext';

const MyComponent = () => {
  const { theme, toggleTheme } = useTheme();
  
  return (
    <div>
      <p>Текущая тема: {theme}</p>
      <button onClick={toggleTheme}>Переключить</button>
    </div>
  );
};
```

### CSS классы

Все стили темной темы применяются через класс `.dark-theme`:

```css
.dark-theme .my-component {
  background-color: var(--bg-primary);
  color: var(--text-primary);
}
```

## Особенности реализации

### 1. Плавные переходы
Все элементы имеют transition для плавной смены цветов:
```css
* {
  transition: background-color 0.3s ease, 
              color 0.3s ease, 
              border-color 0.3s ease;
}
```

### 2. Сохранение состояния
```javascript
// Сохранение
localStorage.setItem('ecology_theme', theme);

// Загрузка
const savedTheme = localStorage.getItem('ecology_theme');
```

### 3. Применение к body
```javascript
if (theme === 'dark') {
  document.body.classList.add('dark-theme');
} else {
  document.body.classList.remove('dark-theme');
}
```
