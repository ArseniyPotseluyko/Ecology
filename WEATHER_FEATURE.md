# Функция погоды в заповедниках

## Описание

Добавлена интеграция с API погоды для отображения прогноза на неделю для каждого заповедника с рекомендациями для посещения.

## Возможности

### 1. Прогноз погоды на 7 дней
- Максимальная и минимальная температура
- Осадки (дождь/снег)
- Скорость ветра
- Иконки погоды
- Описание погодных условий

### 2. Рекомендации для посещения
Система автоматически анализирует прогноз и предоставляет:

**Лучшие дни для посещения:**
- Комфортная температура (15-28°C)
- Минимальные осадки (< 2мм)
- Умеренный ветер (< 20 км/ч)

**Дни требующие осторожности:**
- Сильный дождь (> 10мм)
- Жаркая погода (> 30°C)
- Холодная погода (< 5°C)
- Сильный ветер (> 30 км/ч)

**Общие советы:**
- Рекомендации по одежде
- Необходимые принадлежности
- Предупреждения о погодных условиях

## Технические детали

### API
Используется бесплатный Open-Meteo API:
- Не требует регистрации или API ключа
- Предоставляет точные метеорологические данные
- Обновляется каждые несколько часов

### Backend Endpoint
```
GET /api/reserves/<reserve_id>/weather
```

**Ответ:**
```json
{
  "reserve_name": "Беловежская пуща",
  "forecast": [
    {
      "date": "2024-01-15",
      "temp_max": 5,
      "temp_min": -2,
      "precipitation": 0.5,
      "wind_speed": 15,
      "weather_code": 2
    }
  ],
  "recommendations": {
    "best_days": [...],
    "caution_days": [...],
    "general_advice": [...]
  }
}
```

### Frontend Компонент
`src/components/WeatherWidget.js` - React компонент для отображения погоды

**Особенности:**
- Адаптивный дизайн
- Анимации при наведении
- Поддержка русского и английского языков
- Автоматическое обновление при смене заповедника

## Использование

Виджет погоды автоматически отображается на странице каждого заповедника после секции с животными.

## Коды погоды (WMO)

- 0: Ясно
- 1-3: Облачность
- 45-48: Туман
- 51-67: Дождь
- 71-77: Снег
- 80-82: Ливень
- 95-99: Гроза
