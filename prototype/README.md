# Прототип главной страницы "Экология и мир"

## Описание

Это начальный прототип главной страницы проекта, созданный на чистом HTML/CSS/JavaScript без использования фреймворков. Прототип демонстрирует первоначальную концепцию дизайна и структуры сайта.

## Особенности прототипа

### Реализованные элементы:
- ✅ Базовая структура страницы
- ✅ Фиксированная шапка с навигацией
- ✅ Главный баннер (Hero section)
- ✅ Раздел "О проекте"
- ✅ Сетка карточек заповедников
- ✅ Контактная форма
- ✅ Подвал сайта
- ✅ Плавная прокрутка
- ✅ Базовая адаптивность
- ✅ Простая анимация появления элементов

### Отличия от финальной версии:
- ❌ Нет React компонентов
- ❌ Нет интерактивной карты Leaflet
- ❌ Нет модальных окон с детальной информацией
- ❌ Нет интеграции с backend API
- ❌ Нет продвинутых анимаций
- ❌ Нет темной темы
- ❌ Нет мультиязычности
- ❌ Простой дизайн без сложных эффектов

## Запуск прототипа

Просто откройте файл `index.html` в браузере. Никаких дополнительных установок не требуется.

```bash
# Для локального сервера (опционально):
python -m http.server 8000
# Затем откройте http://localhost:8000
```

## Структура файлов

```
prototype/
├── index.html         # Главная страница
├── about.html         # Страница "О нас"
├── ecology.html       # Страница "Экология"
├── style.css          # Общие стили

├── ecology-page.css   # Стили страницы "Экология"

├── ecology-script.js  # JavaScript для страницы "Экология"
└── README.md          # Документация прототипа
```
