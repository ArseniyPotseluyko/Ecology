## Технические детали

### Источник данных
Описания животных автоматически загружаются из русской Википедии при первом запуске приложения.

### База данных
Таблица `animals` содержит:
- `id` - уникальный идентификатор
- `reserve_id` - ID заповедника
- `name` - название на русском
- `species` - латинское название
- `description` - описание из Википедии (до 800 символов)

### API Endpoint
```
GET /api/reserves/<reserve_id>/animals
```

## Обновление базы данных

Чтобы применить изменения:

1. Удалите существующую базу данных:
```bash
rm backend/ecology.db
```

2. Перезапустите backend:
```bash
cd backend
python app.py
```

База данных будет создана заново с расширенным списком животных.

## Отображение на сайте

Животные отображаются на странице каждого заповедника в секции "Животный мир заповедника" с:
- Названием на русском
- Латинским названием
- Подробным описанием из Википедии
